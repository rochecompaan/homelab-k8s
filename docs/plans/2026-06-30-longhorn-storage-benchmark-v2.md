# Longhorn Storage Benchmark v2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add and run one GitOps-only Longhorn benchmark using the default `longhorn` StorageClass, then fold the result into the existing storage benchmark v2 artifacts.

**Architecture:** Add a temporary ArgoCD Application that points at a new Longhorn benchmark Kustomize path. The benchmark reuses the existing v2 fio Job shape and only changes the backend name, PVC name, and StorageClass. After the run, capture logs/health, regenerate summaries, update the comparison, and remove the app from the bootstrap bundle so ArgoCD prunes the temporary resources.

**Tech Stack:** Kubernetes YAML, ArgoCD Applications, Kustomize, Longhorn CSI, fio, jq, Python summary script, Git.

## Global Constraints

- Use the existing default `longhorn` StorageClass.
- Do not use `longhorn-sata`; it has `diskSelector: sata` and is not the NVMe-focused path.
- Keep the storage benchmark v2 workload shape: 20Gi PVC, 16G fio file, 5 passes, 60 second measured runtime, 10 second ramp, `numjobs=1`, default `iodepth=16`, and `sync-write-4k` at `iodepth=1`.
- Pin the fio consumer pod to `fordyce`.
- Homelab changes are GitOps-only: do not run `kubectl apply`, `kubectl patch`, `kubectl delete`, or `helm upgrade` against homelab resources.
- Captured artifacts live under `docs/storage-benchmark-v2/`.
- No new automated tests are needed for static GitOps YAML/docs; use Kustomize validation, script execution, and read-only cluster checks instead.
- Do not bypass git signing or hooks.

- [ ] **Step 2: Create the Longhorn fio Job**

Run:

```bash
cat > argocd/homelab/storage-benchmark-v2-longhorn/fio-job.yaml <<'EOF'
apiVersion: batch/v1
kind: Job
metadata:
  name: storage-bench-longhorn-v2-run-001
  namespace: storage-benchmark-v2
  annotations:
    argocd.argoproj.io/sync-wave: "1"
  labels:
    app.kubernetes.io/name: storage-benchmark-v2
    storage.compaan.io/backend: longhorn
spec:
  activeDeadlineSeconds: 14400
  backoffLimit: 0
  template:
    metadata:
      labels:
        app.kubernetes.io/name: storage-benchmark-v2
        storage.compaan.io/backend: longhorn
    spec:
      restartPolicy: Never
      nodeSelector:
        kubernetes.io/hostname: fordyce
      containers:
        - name: fio
          image: nixery.dev/shell/fio/jq/coreutils
          env:
            - name: BACKEND
              value: longhorn
            - name: PASSES
              value: "5"
            - name: FIO_SIZE
              value: 16G
            - name: FIO_RUNTIME
              value: "60"
            - name: FIO_RAMP
              value: "10"
          command:
            - /bin/sh
            - -c
          args:
            - |
              echo "RESULT_HEADER,backend,profile,pass,read_iops,write_iops,read_mib_s,write_mib_s,read_mean_ms,write_mean_ms,read_p95_ms,write_p95_ms,read_p99_ms,write_p99_ms,read_p999_ms,write_p999_ms,errors"

              PASSES="${PASSES:-5}"
              FIO_SIZE="${FIO_SIZE:-16G}"
              FIO_RUNTIME="${FIO_RUNTIME:-60}"
              FIO_RAMP="${FIO_RAMP:-10}"
              timestamp="$(date -u +%Y%m%dT%H%M%SZ)"
              result_dir="/volume/results/${BACKEND}/${timestamp}"
              mkdir -p "${result_dir}"

              fio \
                --name=warmup \
                --filename=/volume/fio-test-file \
                --rw=write \
                --bs=1m \
                --size="${FIO_SIZE}" \
                --ioengine=libaio \
                --iodepth=16 \
                --numjobs=1 \
                --direct=1 \
                --time_based \
                --runtime=30 \
                --group_reporting \
                --output-format=json \
                --output="${result_dir}/warmup.json"

              run_profile() {
                profile="$1"
                pass="$2"
                rw="$3"
                bs="$4"
                rwmixread="${5:-}"
                iodepth="${6:-16}"
                output="${result_dir}/${profile}-pass-${pass}.json"

                fio \
                  --name="${profile}" \
                  --filename=/volume/fio-test-file \
                  --rw="${rw}" \
                  --bs="${bs}" \
                  --size="${FIO_SIZE}" \
                  --ioengine=libaio \
                  --iodepth="${iodepth}" \
                  --numjobs=1 \
                  --direct=1 \
                  --time_based \
                  --runtime="${FIO_RUNTIME}" \
                  --ramp_time="${FIO_RAMP}" \
                  --group_reporting \
                  --output-format=json \
                  ${rwmixread:+--rwmixread="${rwmixread}"} \
                  --output="${output}"

                jq -r --arg backend "${BACKEND}" --arg profile "${profile}" --arg pass "${pass}" '
                  .jobs[0] as $job |
                  [
                    "RESULT",
                    $backend,
                    $profile,
                    $pass,
                    ($job.read.iops // 0),
                    ($job.write.iops // 0),
                    (($job.read.bw_bytes // 0) / 1048576),
                    (($job.write.bw_bytes // 0) / 1048576),
                    (($job.read.clat_ns.mean // 0) / 1000000),
                    (($job.write.clat_ns.mean // 0) / 1000000),
                    (($job.read.clat_ns.percentile["95.000000"] // 0) / 1000000),
                    (($job.write.clat_ns.percentile["95.000000"] // 0) / 1000000),
                    (($job.read.clat_ns.percentile["99.000000"] // 0) / 1000000),
                    (($job.write.clat_ns.percentile["99.000000"] // 0) / 1000000),
                    (($job.read.clat_ns.percentile["99.900000"] // 0) / 1000000),
                    (($job.write.clat_ns.percentile["99.900000"] // 0) / 1000000),
                    ($job.error // 0)
                  ] | @csv | sub("^\\\"RESULT\\\","; "RESULT,")
                ' "${output}"
              }

              for pass in $(seq 1 "${PASSES}"); do
                run_profile seq-write-1m "${pass}" write 1m
                run_profile seq-read-1m "${pass}" read 1m
                run_profile rand-write-4k "${pass}" randwrite 4k
                run_profile rand-read-4k "${pass}" randread 4k
                run_profile randrw-4k-70r30w "${pass}" randrw 4k 70
                run_profile sync-write-4k "${pass}" write 4k "" 1
              done
          volumeMounts:
            - name: data
              mountPath: /volume
      volumes:
        - name: data
          persistentVolumeClaim:
            claimName: longhorn-fio-pvc-v2-run-001
EOF
```

- [ ] **Step 2: Regenerate the final comparison table and Longhorn notes**

Run:

```bash
python3 - <<'PY'
from pathlib import Path

combined_path = Path('docs/storage-benchmark-v2/combined-summary.md')
final_path = Path('docs/storage-benchmark-v2/final-comparison.md')
combined = combined_path.read_text()
final = final_path.read_text()

rows = []
for line in combined.splitlines():
    if not line.startswith('|') or line.startswith('| ---'):
        continue
    cells = [cell.strip() for cell in line.strip('|').split('|')]
    if cells[0] == 'backend':
        header = cells
        continue
    rows.append(dict(zip(header, cells)))

by_profile = {}
for row in rows:
    by_profile.setdefault(row['profile'], []).append(row)

def as_float(row, key):
    return float(row[key])

def rank_line(profile, metric, label):
    ranked = sorted(by_profile[profile], key=lambda r: as_float(r, metric), reverse=True)
    names = [r['backend'] for r in ranked]
    longhorn = next(r for r in ranked if r['backend'] == 'longhorn')
    place = names.index('longhorn') + 1
    return f"- `{profile}` {label}: Longhorn ranked {place}/{len(ranked)} at {longhorn[metric]} ({metric})."

notes = [
    '## Longhorn Add-On Findings',
    '',
    rank_line('seq-read-1m', 'read_mib_s_avg', 'read throughput'),
    rank_line('seq-write-1m', 'write_mib_s_avg', 'write throughput'),
    rank_line('rand-read-4k', 'read_iops_avg', 'read IOPS'),
    rank_line('rand-write-4k', 'write_iops_avg', 'write IOPS'),
    rank_line('sync-write-4k', 'write_iops_avg', 'single-depth write IOPS'),
]
if 'randrw-4k-70r30w' in by_profile:
    notes.append(rank_line('randrw-4k-70r30w', 'read_iops_avg', 'mixed read IOPS'))
    notes.append(rank_line('randrw-4k-70r30w', 'write_iops_avg', 'mixed write IOPS'))
notes.append('')
notes_text = '\n'.join(notes)

table = '\n'.join(line for line in combined.splitlines() if line.startswith('|')) + '\n'

before, rest = final.split('## Performance Summary\n', 1)
_, after_summary = rest.split('## Plain-English Findings\n', 1)
final = before + '## Performance Summary\n\n' + table + '\n## Plain-English Findings\n' + after_summary

marker = '## Longhorn Add-On Findings\n'
if marker in final:
    start = final.index(marker)
    next_recommendation = final.index('## Recommendation\n', start)
    final = final[:start] + notes_text + '\n' + final[next_recommendation:]
else:
    final = final.replace('## Recommendation\n', notes_text + '\n## Recommendation\n', 1)

final_path.write_text(final)
PY
```
